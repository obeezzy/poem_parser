# poem_parser
Parse words into stressed and unstressed syllables.

## Installation
To install:
```bash
$ pip install poem_parser
```

## Example usage
To parse a line:
```bash
$ python -m poem_parser "Shall I compare thee to a summer's day?"
```

To parse a file:
```bash
$ python -m poem_parser -f poem.txt
```
